Mastodon Portal of Web Apps

Mastodon is a software for running self-hosted social networking services in Fediverse. Since these services are decentralized, we often want a gateway of our Mastodon servers. This portal provides just it.

All of web apps of here are fully written by JavaScript / HTML / CSS only. It means you only need accessing hosting servers to install web apps to use them. They do not send your data to unknown servers. If you have doubts, you can read codes of web apps on here. If you have doubts for hosting services, you can fork it and self-host.
